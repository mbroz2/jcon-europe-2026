## Agentic Workflows
- Integrating AI agents into a Quarkus application
- Building autonomous agents that can make decisions and use tools

- Invoking agents in parallel workflows to perform work more efficiently
- Building conditional workflows that let you control which agents work on a request
- Combining agents and workflows of agents into nested workflows
- Using the supervisor pattern for dynamic orchestration
- Engaging remote agents, potentially built using different agentic frameworks, using Agent2Agent (A2A)
